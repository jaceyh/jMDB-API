jMDB is an API that allows users to request movie data.

Users may request data such as genre, director, and cast.
Users may register to create a personalized experience that allows them to add movies to a "Favorites" list.
Request URL Request Method Parameters Request Format Response Format
Get list of movies /movies GET none none JSON object holding data for all movies
Get data for a single movie /movies/[name] GET :title none JSON object holding data for specified movie
Get data for a director /directors/[name] GET :director none JSON object holding data for specified director
Get data for a tag /tags/[tag] GET :tag none JSON object holding data for movies with specified tag
Create user account /users POST none JSON object holding desired username, password, email address, and birthdate

{
"username": "Jane Doe"
"password": "xxxx1234"
"email": "xxx123@x.com"
"birthdate": "1987-02-22"
}

JSON object holding user data with id assigned

{
"id": 13
"username": 'Jane Doe'
"password": "xxxx1234"
"email": "xxx123@x.com"
"birthdate": "1987-02-22"
"favoriteMovies": "[]"
"watchlistMovies": "[]"
}

Update User Info /users/[id] PUT :id JSON object holding user data

{
"username": "Jane Doe"
}

JSON object containing updated user data

{
"id": "13"
"username": "Jane Doe"
"favoriteMovies": "[]"
"watchlistMovies": "[]"
}

Delete User Account /users/delete/[id] DELETE :id none Text response indicating user has been deleted.
Add Movie to Favorites /users/[id]/favorites/[movie title] PUT :id, :title none Text response indicating movie was added to your favorites.
Add Movie to Watchlist /users/[id]/watchlist/[movie title] PUT :id, :title none Text response indicating movie was added to your watchlist.
Remove Movie from Favorites /users/[id]/favorites/delete/[movie title] DELETE :id, :title none Text response indicating movie was removed from your favorites.
Remove Movie from Watchlist /users/[id]/watchlist/delete/[movie title] DELETE :id, :title none Text response indicating movie was removed from your watchlist.
