jMDB is an API that allows users to request movie data.
Users may request data such as genre, director, and cast. Users may register to create a personalized experience that allows them to add movies to a "Favorites" list.
Request URL Request Method Parameters Request Format Response Format
Get list of movies /movies GET none none JSON object holding data for all movies
Get data for a single movie /movies/[title] GET :title none JSON object holding data for specified movie
Get data for a director /movies/[director] GET :director none JSON object holding data for specified director
Get data for a director /movies/[Director] GET none none JSON object
